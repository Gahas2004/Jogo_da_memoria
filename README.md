# Jogo da Memória

## Descrição

Este é um simples Jogo da Memória desenvolvido com HTML, CSS e JavaScript. O objetivo do jogo é encontrar todos os pares de cartas.

## Regras do Jogo

1. Encontre todos os pares de cartas.
2. Clique em uma carta para virá-la.
3. Se duas cartas consecutivas forem iguais, elas permanecerão viradas.
4. Caso contrário, elas serão viradas de volta.
5. O jogo termina quando todos os pares forem encontrados.
